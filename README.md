# Github練習用リポジトリ
## Githubをいろいろいじって遊んでみる
  
## pull requestの流れ
1. ローカルにない場合、Githubからリポジトリファイルをcloneします。
2. 作業ディレクトリに移動します。
3. 作業ブランチがmasterの場合、ブランチを切ります。
4. 作業ブランチへ切り替えてmasterの更新に追随します 
5. 作業します。
6. Githubにブランチをpushします。
7. Github上でmasterへpull requestを送ります。
